# APL exemplar — the original 253 patterns (factual index)

Christopher Alexander et al., *A Pattern Language: Towns, Buildings, Construction* (OUP, 1977). Factual index — names, numbers, groupings — for use as the canonical exemplar of scale structure. Do not reproduce the book's prose; point users to the book or https://patternlanguage.cc/.

## Original conventions worth imitating

- Three scale tiers: **Towns** (1–94, grown piecemeal by communities), **Buildings** (95–204, designable by individuals/small groups), **Construction** (205–253, buildable directly).
- Within tiers, thematic clusters of 4–10 patterns, each introduced by one connective instruction sentence — the model for `index.md` group prose.
- Confidence asterisks: ✻✻ invariant / ✻ progress / none = hypothesis.
- Page-form: photograph → context paragraph ("…this pattern completes…") → **bold problem** → evidence body → "Therefore:" **bold instruction** → diagram → downstream links.
